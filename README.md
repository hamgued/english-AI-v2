# English & AI — Interactive Learning Tools

Interactive teaching tools for AI literacy, built for the **English & AI** course at [LAB University of Applied Sciences](https://lab.fi/en).

## Modules

| Module | File | Topic |
|--------|------|-------|
| 1 | `ai-history-concepts-metaphors.html` | AI history, core concepts, and common metaphors |
| 2 | `ai-risks-ethics.html` | Hallucinations, bias, environmental impact, copyright, and ethical frameworks |
| 3 | `ai-writing-prompting.html` | Using GenAI in academic writing and prompting techniques |

## Hosting

Plain HTML — no build step. Deploy via GitHub Pages (Settings → Pages → main branch, root folder).
